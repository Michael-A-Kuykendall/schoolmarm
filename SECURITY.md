# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in SchoolMarm, please report it responsibly.

### How to Report

**DO NOT** open a public GitHub issue for security vulnerabilities.

Instead, please email: **michaelallenkuykendall@gmail.com**

Include:
- A description of the vulnerability
- Steps to reproduce it
- Potential impact
- Any relevant logs or output

We will acknowledge receipt within 48 hours and provide a timeline for triage and resolution.
